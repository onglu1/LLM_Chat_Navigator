# ChatGPT Scrollmap

一个 Chrome 扩展，在 ChatGPT 页面右侧添加侧边栏，展示当前对话中的所有提示词（Prompt），方便快速浏览和跳转。

## 功能

- **侧边栏展示**：以卡片形式列出当前对话的所有用户提示词，显示序号和内容预览
- **快速跳转**：点击卡片即可滚动到对应的提示词位置
- **上下导航**：通过悬浮按钮在提示词之间快速切换，上一个按钮支持"先回顶再跳转"的音乐播放器式逻辑
- **分支信息**：当对话存在分支时，卡片底部显示分支编号
- **展开/折叠**：一键切换侧边栏显示状态，折叠后不占用页面空间
- **自动更新**：每 2 秒自动刷新提示词列表，实时跟踪对话变化

## 安装

1. 下载或克隆本仓库
2. 打开 Chrome，进入 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本项目文件夹

## 使用

安装后访问 [chatgpt.com](https://chatgpt.com)，页面右侧会出现三个悬浮按钮：

| 按钮 | 功能 |
|------|------|
| 侧边栏 | 展开/折叠提示词侧边栏 |
| ▲ | 跳转到上一个提示词 |
| ▼ | 跳转到下一个提示词 |
